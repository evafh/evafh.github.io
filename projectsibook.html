SiBook (Sistem Informasi Toko Buku Online)

SiBook is an online bookstore designed like an e-commerce website, exclusively for purchasing books. Used Django framework for backend and Bootstrap for frontend. Key features include register and login, homepage and book catalog, book details, search books, shopping cart, checkout, and check order status. Designed prototype website (high fidelity), user experience, problem statement, and created use case
